HTML 

1.  HTML이란? (Hyper Text Markup Language)
-웹페이지를 만드는 컴퓨터 언어 




2. 특징 
- 쉽다 
- 중요하다 
- html 파일이라는 표시를 남겨주어야 한다. <!DOCTYPE html>




3. 문법 
   **tag 
      -- 표현이나 어떤 기능의 수행을 지시하는 짧은 낱말 
      --
      -- 코드 작성 방법 : <></>
      -- 열리는 태그, 닫히는 태그라고 부른다 (두 개를 구분하기 위해 닫히는 태그 앞에 /를 붙인다)
      -- 자주 사용하는 태그 
	<strong></strong> : 글씨 굵게 
	<h1~h6></h1~h6> : 제목을 나타냄 , h 뒤에 오는 숫자가 커질수록 글씨 크기는 작아진다. 
	<br>                      : 줄바꿈 태그 , 닫히는 태그를 사용하지 않아도 된다.     
	<p></p>                : 단락을 나타내는 태그   
	<img src="" >         : 사진을 추가할 수 있음. src라는 속성을 통해 사진의 주소를 적어야한다. 
	<li></li>                 : 목록을 나타낼 때 사용(자식태그). 
	<ul></ul>               : Unordered List
			 정렬되지 않은 목록을 그룹지을 때 사용(부모태그) , 
			 목록 앞에 검은색 점 기호가 생성됨.
	<ol></ol>              : Ordered List
			 순서가 있는 목록을 그룹지을 때 사용(부모태그) , 
			 목록 앞에 기호가 숫자로 표시되며 저장해둔 목록을 추가, 삭제 할 시 자동적으로 숫자가 넘버링됨. 
	<title></title>         : 웹 페이지의 제목을 나타냄. 
	<meta></meta>     : 웹 브라우저에서 html 파일을 열 때 저장된 파일의 형식으로 띄워주기 위해 사용됨.  
	<body></body>     : 본문 
	<head></head>     : 본문을 설명함
	<html></html>       : html 문서의 루트요소임을 나타냄.
			**루트 요소란 : html 문서의 최상단 요소. 최고 조상이라고 생각하면 편할듯. 
			 
	<!DOCTYPE html>  : 웹 문서의 유형을 html로  지정(이 문서는 html이다) 
			 html 파일의 제일 상단에 작성한다.
	<a></a>               : 웹과 웹을 이어주는 링크태그  
			<a href=" " target="_blank" title=" " >
			href : 가져올 웹의 주소를 적어 연결해주는 속성 
			         Hypertext reference 의 줄임말 
			        이 링크는 이 값을 참조한다는 뜻.
			         
			target="_blank" :  걸어둔 링크가 새창으로 열리는 속성
			title :  툴팁 속성 
				툴팁 ? 커서 or 마우스포인터와 함께 사용하여 항목을 클릭하지 않고도 항목에 대한 정보를 표시하는 그래빅 사용자 인터페이스(GUI) 
			
	
	           
      --부모 ,자식 태그 
	기본 사용 형식 
	<부모태그>
		<자식태그></자식태그>
	</부모태그>
	
	ex) 
	<ul>                      :  부모태그임. 반드시 자식태그가 필요함.

	       <li></li>          :  자식태그임.
			 li태그의 내용들을 한 그룹으로 묶어주기 위해 반드시 부모태그가 필요함 
	</ul>
	





   **속성
      --  속성을 통해 태그의 기본 기능을 수정 및 특정한 기능을 추가할 수 있다. 
	<img src="Samoyed.jpg" width="300px">
	: 이미지 태그 안에 업로드할 사진의 주소는 src라는 속성을 통해, 사진의 크기는 width를 통해 
	   그 밖의 많은 속성들을 통해 부족한 태그를 채워 줄 수 있다.
